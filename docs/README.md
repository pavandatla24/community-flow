# Community Flow

A real-time “wellness weather report” for Chicago, starting with an English-only MVP.

This repo contains:

- `scraper/` – collects public wellness text
- `nlp/` – NLP pipeline (themes + clusters)
- `backend/` – FastAPI API
- `frontend/` – React UI
- `automation/` – GitHub Actions
- `docs/` – documentation
- `data/` – raw + processed data (not committed in production)
